# Welcome to Sunny Stack

Sunny Stack is a batteries-included full-stack TypeScript SaaS starter that combines the best modern tools into a cohesive, production-ready foundation. Built for developers who want end-to-end type safety without sacrificing flexibility.

## Key Features

### End-to-End Type Safety
Full TypeScript coverage from database to UI with tRPC bridging frontend and backend. No manual type synchronization or code generation needed.

### Production-Ready Authentication
Better Auth integration provides secure, open-source authentication with OAuth support, session management, and zero vendor lock-in.

### Modern Backend Architecture
NestJS enforces clean, modular structure with dependency injection, while tRPC handles type-safe API calls. Traditional REST endpoints available for webhooks and external integrations.

### Powerful Database Layer
Drizzle ORM offers SQL-like syntax with full type safety, paired with PostgreSQL for reliability and advanced features like PGVector.

### Optimized Frontend
Tanstack Router provides file-based routing with type-safe navigation, while Tanstack Query handles server state management with caching and mutations.

### Customizable UI Components
Shadcn UI components live in your codebase, not hidden behind packages. Customize everything while maintaining rapid development speed.

### Monorepo Architecture
PNPM workspaces enable shared code between frontend and backend, streamlined dependency management, and faster installs.

<Tip>
Sunny Stack is opinionated by design. These constraints help maintain clean architecture as your application scales from MVP to production.
</Tip>

## Philosophy

This stack is built on the principle that **opinionated structure leads to better long-term maintainability**. Rather than stitching together disparate tools, Sunny Stack provides a cohesive foundation where each technology complements the others.

The architecture separates frontend and backend concerns while maintaining seamless type safety. This separation is ideal for client-side rendered SaaS applications that need the flexibility to scale independently.

<Note>
All code is yours to modify. Unlike SaaS platforms or opaque packages, you have complete control over every aspect of your application.
</Note>
